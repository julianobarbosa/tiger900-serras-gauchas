### Como usar os GPX

**Garmin:**
Copie os arquivos `.gpx` para `/Garmin/GPX/` e sincronize com BaseCamp.

**My Triumph App:**
Abra `assets/Links_Triumph_MyApp.txt` e envie as rotas via Bluetooth para o painel TFT.

## Rotas de Retorno

Voce tem **duas opcoes** de retorno a partir de Bento Goncalves:

### Opcao 1: Retorno Rapido (Padrao)
```
Bento → Vacaria → Curitiba → Ourinhos → Uberaba → Goiania
```
- ~1.570 km em 5 dias
- Estradas melhores, mais rapido

### Opcao 2: Retorno Cenico
```
Bento → Sao Jose dos Ausentes → Vacaria → Lages → Curitiba → Goiania
```
- ~1.700 km em 5 dias
- Paisagens incriveis, revisita a serra

**Decida durante a viagem conforme seu estado de animo!**
