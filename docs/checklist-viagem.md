# Checklist de Viagem - Serras Gaúchas 2026

## Documentos e Dinheiro

### Obrigatórios

- [ ] CNH válida (categoria A)
- [ ] CRLV da moto (documento do veículo)
- [ ] Seguro da moto (apólice ou app)
- [ ] Cartão do plano de saúde
- [ ] RG ou CNH (documento pessoal)
- [ ] Cartões de crédito/débito (pelo menos 2 bandeiras)
- [ ] Dinheiro em espécie (R$ 500-1.000 para emergências)
- [ ] PIX configurado no celular

### Recomendados

- [ ] Cópia dos documentos (física ou digital)
- [ ] Contato de emergência anotado
- [ ] Número do seguro/assistência 24h
- [ ] Reservas de hospedagem (impressas ou no celular)

---

## Equipamentos de Segurança (Piloto)

### Obrigatórios

- [ ] Capacete (certificado Inmetro)
- [ ] Luvas (impermeáveis de preferência)
- [ ] Jaqueta com proteções (ombro, cotovelo, costas)
- [ ] Calça com proteções (joelho, quadril)
- [ ] Botas cano alto (acima do tornozelo)

### Recomendados

- [ ] Protetor de coluna separado (se jaqueta não tiver)
- [ ] Colete refletivo (para noite/neblina)
- [ ] Viseira extra (fumê + cristal)
- [ ] Pinlock anti-embaçante
- [ ] Balaclava (frio nas serras)
- [ ] Protetor de pescoço (neck tube)

## Roupas (20 dias - Clima Variável)

### Para o Frio das Serras (0-15°C)

- [ ] Segunda pele térmica (2 conjuntos)
- [ ] Fleece/soft shell (2 unidades)
- [ ] Jaqueta corta-vento (para usar sob a jaqueta de moto)
- [ ] Meias térmicas (3 pares)
- [ ] Gorro/touca
- [ ] Luvas térmicas finas (para usar sob as luvas de moto)
- [ ] Cachecol ou buff

### Para Dias Quentes (20-30°C)

- [ ] Camisetas dry-fit (5 unidades)
- [ ] Bermudas/shorts (3 unidades)
- [ ] Calça leve (2 unidades)
- [ ] Chinelo/sandália (para descanso)

### Proteção contra Chuva

- [ ] Capa de chuva completa (calça + jaqueta)
- [ ] Cobre-bota impermeável
- [ ] Luvas impermeáveis (ou capa para luvas)
- [ ] Saco estanque para roupas

### Roupas Gerais

- [ ] Cuecas/roupas íntimas (7 unidades - lavar no caminho)
- [ ] Meias normais (5 pares)
- [ ] Pijama/roupa de dormir
- [ ] Tênis casual (para passeios)

## Dicas Finais

1. **Teste tudo antes:** Monte a barraca, use o fogareiro, carregue a moto
2. **Menos é mais:** Na dúvida, não leve
3. **Roupas:** Prefira camadas a peças pesadas
4. **Eletrônicos:** Sempre em sacos impermeáveis
5. **Documentos:** Cópias em nuvem (Google Drive, iCloud)
6. **Dinheiro:** Distribua em lugares diferentes
7. **Pneus:** Verifique pressão toda manhã
