# Visão Geral do Projeto

## Triumph Tiger 900 Rally Pro – Serras Gaúchas Adventure 2026

### Resumo Executivo

Repositório de planejamento e navegação para uma viagem de moto de 20 dias pelas Serras Gaúchas, partindo de Goiânia-GO. Contém rotas GPS, planilha de custos e materiais de apoio para integração com dispositivos Garmin e o aplicativo My Triumph.

### Informações Gerais

| Atributo | Valor |
|----------|-------|
| **Tipo de Projeto** | Repositório de Dados de Viagem |
| **Período da Viagem** | 19/01/2026 → 02/02/2026 |
| **Duração** | 20 dias |
| **Distância Estimada** | ~3.500 km |
| **Veículo** | Triumph Tiger 900 Rally Pro 2025 |
| **Autor** | Juliano M. Barbosa |

### Stack Tecnológico

| Categoria | Tecnologia | Propósito |
|-----------|------------|-----------|
| Rotas GPS | GPX 1.1 | Waypoints e tracks para Garmin |
| Planilha | XLSX (Excel) | Custos, hospedagem, abastecimento |
| Documentação | Markdown | READMEs e guias |
| Navegação | Google Maps URLs | Links compartilháveis |
| Integração | My Triumph App | Sincronização com painel TFT |
