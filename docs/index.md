## Começando

### Para usar as rotas no GPS Garmin:

1. Copie os arquivos `.gpx` para `/Garmin/GPX/`
2. Sincronize com BaseCamp
3. As rotas aparecerão no dispositivo

### Para usar no My Triumph App:

1. Abra os links em `assets/Links_Triumph_MyApp.txt`
2. Compartilhe via Bluetooth com a moto
3. A rota aparecerá no painel TFT

### Para consultar custos:

1. Abra `assets/Roteiro_Tiger900_SerrasGauchas.xlsx`
2. Consulte hospedagens, alimentação e combustível por dia
