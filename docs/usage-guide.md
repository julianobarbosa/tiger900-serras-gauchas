# Guia de Uso

## Como Usar os Arquivos GPX

### Garmin (BaseCamp)

1. Conecte seu GPS Garmin ao computador via USB
2. Abra o Garmin BaseCamp
3. Importe os arquivos `.gpx` da pasta `Rotas_Tiger900RallyPro/`
4. Arraste as rotas para o dispositivo conectado
5. Desconecte o GPS e as rotas estarão disponíveis

**Alternativa manual:**
1. Conecte o GPS via USB
2. Navegue até a pasta do dispositivo: `/Garmin/GPX/`
3. Copie os arquivos `.gpx` diretamente
4. Ejete o dispositivo com segurança

### My Triumph App (Painel TFT)

1. Abra o arquivo `assets/Links_Triumph_MyApp.txt`
2. Copie o link da rota desejada
3. Abra o link no navegador do celular
4. Use a opção "Compartilhar" → "My Triumph"
5. Conecte o celular via Bluetooth à moto
6. A rota aparecerá no painel TFT

### Outros Apps (OsmAnd, Kurviger, etc.)

1. Copie os arquivos `.gpx` para o celular
2. Abra o app de navegação
3. Importe o arquivo GPX
4. Siga a rota no app

## Como Usar a Planilha de Custos

### Visualização

Abra `assets/Roteiro_Tiger900_SerrasGauchas.xlsx` com:
- Microsoft Excel
- LibreOffice Calc
- Google Sheets (upload)
- Numbers (macOS)

### Estrutura da Planilha

A planilha contém:
- **Dia a dia**: Data, cidade, hospedagem, custo
- **Abastecimento**: Postos sugeridos, distâncias
- **Totais**: Soma por categoria

### Personalizando

1. Faça uma cópia da planilha
2. Ajuste valores conforme sua realidade
3. Adicione/remova dias se necessário
4. Atualize os totais

## Links Rápidos do Google Maps

### Usar no Celular

1. Abra `assets/Links_Triumph_MyApp.txt`
2. Clique/toque no link desejado
3. O Google Maps abrirá com a rota
4. Toque em "Iniciar" para navegar

### Compartilhar

Os links são públicos e podem ser compartilhados via:
- WhatsApp
- Email
- Qualquer mensageiro

## Antes da Viagem

### Checklist de Preparação

- [ ] Carregar arquivos GPX no GPS Garmin
- [ ] Testar sincronização com My Triumph App
- [ ] Imprimir planilha de custos (backup)
- [ ] Salvar links offline no celular
- [ ] Verificar reservas de hospedagem
- [ ] Revisar moto (óleo, pneus, freios, corrente)

### Equipamentos Recomendados

- GPS Garmin com suporte para moto
- Celular com My Triumph App instalado
- Carregador 12V para moto
- Power bank para emergências
- Capa de chuva e equipamento para frio (serras)

## Durante a Viagem

### Navegação

Use a rota de **ida cênica** (`02_Rota_Ida_Cenica.gpx`) para aproveitar as paisagens na ida.

Use a rota de **volta asfaltada** (`03_Rota_Volta_Asfaltada.gpx`) para um retorno mais rápido e seguro.

### Atualizando Custos

Anote os gastos reais na planilha para comparar com o estimado ao final.
